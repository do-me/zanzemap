Mosquito Activity

Mosquito Activities for Regions and Municipalities

About This Application

This interactive dashboard visualizes weekly mosquito activity data across Italy. The system helps public health officials and researchers track mosquito activity patterns to better predict and prevent disease outbreaks.

The data represents mosquito egg count probabilities, with higher values indicating greater mosquito activity. Data is collected weekly from April to November.

Data Visualization

Use the map to explore mosquito activity across regions:

  • Select a thematic layer to display on the map.
  • Adjust the timeline slider to see weekly changes for the mosquito layer.
  • Click on any region to view its detailed time series data.

How to Use

Interactive Chart: The chart at the bottom shows time series data. Click on any region on the map to see its specific data.

Data Selection:

  • Single region: Click on a polygon
  • All regions: Reset selection
